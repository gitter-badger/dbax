Why DBAX?

PL/SQL MVC Web Framework

DBAX is the first MVC PL/SQL framework. Take the control of your application code (front-end and back-end).

Web console

Configure your application, review logs, create users... through an intuitive Web console.

Seceure

Simple authentication, RBAC for Authorization and secure coding tools that you need to do things right

Excepcional performance

Engineered with performance tuning in mind.

Web standards

HTML5, CSS3 and Responsive Design, JavaScript and their libraries (AngularJS, ReactJS)...

Fast development

Live changes, scaffolding, user session state, simple url routing, code generators (tapiGen2)...

Open Source

DABX is a Open Source project.

Enterprise ready

Scale your Oracle DataBase. Oracle RAC, Oracle Cloud, result cache, Native compilation...
